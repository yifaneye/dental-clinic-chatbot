## Prerequisites
You need:
1. git
2. Python3
3. [Docker](https://www.docker.com/get-started)
4. [Docker compose](https://docs.docker.com/compose/)
5. npm
6. ["serve" package](https://www.npmjs.com/package/serve)

## Getting Started

Run the following command:

```
sh runme.sh
```

Go to [http://localhost:5004](http://localhost:5004) to meet the chatbot
